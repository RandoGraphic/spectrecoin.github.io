PROJECT KEY FEATURES

IN-CLIENT INTEGRATED TOR

Built in-client TOR ([The Onion Router] www.torproject.org) hides users IP by routing traffic through several hops on the TOR network. So that the SpectreCoin client runs as a Tor hidden service using a .onion address to connect to other clients in the network. Your real IP address is therefore protected at all times.

ANONYMOUS COIN CREATION

Through the use of dual-key stealth technology SpectreCoin provides the ability to generate ‘anonymous coins’ that we call SPECTRE that can be spent in anonymous and private transactions through the use of Ring Signatures. SpectreCoin is a dual coin system and also comprises the 'normal' coin we call XSPEC that can be used in traditional transactions much like Bitcoin and most other cryptocurrencies.

RING SIGNATURES + STEALTH ADDRESSES

CryptoNote inspired protocol provides dual-key stealth addresses and ring signatures to further anonymize the transactions conducted with SPECTRE (anonymous coins).
Ring Signatures is a type of digital signature that can be performed by any member of a group of users. Makes your transactions untraceable for blockchain explorer.
SPECTRE can be sent anonymously through an implementation of Ring Signatures based on the protocol to eliminate any transaction history. The wallet offers the opportunity to transfer your balance between public coins(XSPEC) and ‘anonymous coin’ (SPECTRE). We are currently working on improving this technology to improve functionality and privacy.

PROOF-OF-STAKE V3

Proof-of-Stake version 3 is blockchain consensus algorithm, where the creator of the next block is chosen via various combinations of random selection and wealth or age (i.e., the stake), making transactions approval fast and a network more scalable. You can learn more information here.
